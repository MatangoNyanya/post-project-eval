# 🧭 EvalNavi — JICA事業評価ナレッジ活用システム

## プロジェクト概要

JICAの過去2,000件の事後評価報告書（2010〜2025年度）を分析した機械学習モデルに基づき、新規案件のリスクスクリーニングと過去の教訓提示を行うデモ用モックアップWebアプリケーション。

**本番同等のUX・デザイン**を実現しつつ、**すべてダミーデータ＆ルールベースロジックで動作**（実MLモデル不使用）。

---

## 実装済み機能

### STEP 1: 案件情報入力
- 対象国セレクトボックス（21カ国・5地域）
- 国名選択に応じた地域の自動入力
- セクター選択に応じたサブセクターの動的更新（9セクター）
- スキーム選択（ラジオボタン）
- 計画事業費（数値入力）
- 計画事業期間（スライダー、6〜120ヶ月）
- 事業概要テキストエリア
- EvalNaviの仕組み・分析基盤情報パネル（右カラム）

### STEP 2: 予測結果 + SHAP分析
- プログレスバー付きローディングアニメーション（2秒）
- 入力値サマリーバッジ表示
- 半円形リスクゲージチャート（ECharts Gauge）
  - 0〜100%、3段階カラーゾーン（緑/オレンジ/赤）
- リスク判定テキスト（3段階: 良好/要注意/危険）
- 評価項目別予測カード（妥当性/有効性/効率性/持続性）
- SHAP横棒チャート（ECharts Bar）：要因別影響度の可視化
- SHAP要因別説明テキストリスト
- 類似案件テーブル（10件）：評価グレード別カラーコーディング
- 類似案件統計サマリー（A/B/C/D件数・割合）

### STEP 3: リスク要因別 教訓アドバイス
- プログレスバー付きローディングアニメーション（3秒）
- リスク要因別アコーディオンパネル（展開/折りたたみ）
  - デフォルト：最大リスク要因のみ展開
- 各パネル内容：
  - 教訓テキスト（引用スタイル）
  - 出典タグ（事業名・年度・ページ番号）
  - 成功パターンリスト
  - 統計補足バーチャート（対応するデータがある場合）
- 総合サマリーカード（注意点3ポイント）
- アクションボタン群（PDF出力・詳細一覧は本番版予定、再分析は有効）

### ダミーロジック
- 地域・セクター・スキーム・予算・期間に基づくルールベースリスク計算
- 入力条件に応じた3パターンの教訓データ切り替え：
  1. デフォルト（ラオス×道路×技協）
  2. サブサハラアフリカ×水資源・防災
  3. 南アジア×教育

### UI/UX
- サイドバー（システム情報・モデル情報・免責事項）
- ステップインジケーター（完了ステップはチェックマーク表示）
- レスポンシブ対応（1024px以上推奨、モバイルも動作）
- スムーズなフェードインアニメーション

## UI/UXとモダンデザインの導入 (2026-03-29更新)
- **Glassmorphism:** ヘッダーやローディングオーバーレイに `backdrop-filter: blur` を採用。
- **Modern Color Palette:** トラディショナルな単色ベタ塗りから、スレートブルー（深みのある青）とスカイブルーを基調としたグラデーションや、最新のTailwind系カラーパレットへ刷新。
- **Soft Shadows & Rounded Corners:** より柔らかく広がるドロップシャドウ (`box-shadow`) と、角丸 (`border-radius: 16px` など) を採用して今風のカードUIに改善。
- **Subtle Background Gradients:** 背景を単なるグレーから、わずかに色味の入ったRadial Gradient（放射状グラデーション）へ変更。
- **Hover & Active Animations:** ボタンやカードへのホバー時にふわっと浮かび上がる `transform: translateY(-2px)` と、ステップインジケーターのパルスアニメーションなど、マイクロインタラクションを追加。

### 旧デザインからの主な変更点
- JICA青カラーパレットから、よりモダンでクリーンなSaaS風UIデザインへスタイル（`css/style.css`）を刷新。
- サイドバーや各コンポーネントの余白（Padding / Margin）を調整し、視認性を向上。

## デモシナリオ（推奨操作手順）

### シナリオ1: デフォルト（ラオス×道路×技協）
1. デフォルト値のまま「スクリーニングを実行する」
2. 要注意確率〜68%台、SHAP要因を確認
3. 「教訓・対策を確認する」→ リスク要因1（ラオス×制度基盤）を展開

### シナリオ2: 高リスク（タンザニア×上水道×有償）
1. 「条件を変えて再分析」でSTEP1に戻る
2. タンザニア、水資源・防災、上水道、有償資金協力、150億円
3. より高いリスク確率 + アフリカ水セクター向け教訓が表示

### シナリオ3: 低リスク（ベトナム×教育×技協）
1. ベトナム、教育、高等教育、技術協力、3億円、36ヶ月
2. 低いリスク確率 → 「良好な評価が見込まれます」
